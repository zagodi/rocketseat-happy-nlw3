## <p align="center">Instalação</p>

* Backend

```
$ git clone https://github.com/zagodi/rocketseat-happy-nlw3

$ cd rocketseat-happy-nlw3
$ cd backend

$ yarn install

$ yarn dev
```


* Frontend

```
$ git clone https://github.com/zagodi/rocketseat-happy-nlw3

$ cd rocketseat-happy-nlw3
$ cd nlw-web

$ yarn install

$ yarn start
```

* Mobile

```
$ git clone https://github.com/zagodi/rocketseat-happy-nlw3

$ cd rocketseat-happy-nlw3
$ cd mobile

$ yarn install

$ yarn start
```
